# 🚀 AlgoYou  
**“Don’t let your algorithm control you — take control of your own algorithm.”**  

—

## 📖 Description  
**AlgoYou** is a Chrome extension that gives users insight into their YouTube activity.  
It tracks what you watch and search, then generates a **simple, bullet-pointed memo** that highlights patterns and shows how algorithms may be shaping your recommendations.  

👉 The goal is to put users back in charge of their digital experience.  

## ✨ Features  
- 🔎 **Tracks YouTube activity** (watch/search history)  
- 📝 **Generates easy-to-read summaries** of your content consumption  
- 📊 **Highlights habits and viewing patterns**  
- 🌟 **Future vision**: delete/edit stored data, add chatbot navigation, and provide feedback on harmful content  

## 🛠️ How to Run Locally  
1. Go to `chrome://extensions/` in your Chrome browser  
2. Toggle **Developer mode** (top right)  
3. Click **Load unpacked** → select the downloaded folder  

🎉 **AlgoYou should now appear in your Chrome extensions!**
